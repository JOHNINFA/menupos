# 📚 Documentación de MenuPOS

> Tu **curso personal** mientras construyes MenuPOS.
> Material organizado en 3 capas para que aprendas de verdad.

---

## 🗂️ Estructura

### 📖 [`clases/`](clases/) — Mini-clases por concepto
Cada concepto técnico (Modelos, Serializers, JWT, S3, etc.) tiene su propia clase corta con:
- 🤔 El problema que resuelve
- 🎬 Una analogía simple (metáfora del restaurante)
- 💻 Ejemplo de código
- 🧠 Quiz rápido
- 🔗 Link a documentación oficial

### 📊 [`diagramas/`](diagramas/) — Diagramas visuales
Visualizaciones con **Mermaid** (renderizan automáticamente en GitHub):
- Arquitectura general
- Modelo Entidad-Relación
- Flujos de petición HTTP
- Flujo de autenticación

### 🧠 [`quizzes/`](quizzes/) — Quizzes por fase
Al final de cada fase del proyecto, un quiz para verificar que entendiste lo importante.

## 🎯 Cómo aprovechar el material

1. **Lee la clase** completa antes de tocar código de ese concepto
2. **Mira el diagrama** relacionado para tener la imagen mental
3. **Cuando escribamos el código**, vuelve a la clase si te pierdes
4. **Al final de la fase**, haz el quiz
5. Si fallas algo del quiz → relee la clase correspondiente
