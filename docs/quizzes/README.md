## 🎯 Cómo hacer los quizzes

1. Lee la pregunta
2. **Intenta responder SIN ver el código** (es lo que pasa en entrevistas)
3. Compara con la respuesta correcta
4. Si fallaste, vuelve a la clase relacionada y reléela
5. Repite el quiz hasta tener 100%
